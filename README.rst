Install
------------------------------------------------------------------------------

``seedinvest_monitor`` is released on PyPI, so all you need is:

.. code-block:: console

    $ pip install seedinvest_monitor

To upgrade to latest version:

.. code-block:: console

    $ pip install --upgrade seedinvest_monitor
